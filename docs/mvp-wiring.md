# 灵巧手指 MVP 舵机版接线图

## 1. 当前硬件版本

这份接线图按当前舵机 MVP 来写：

1. 主控：ESP32-C3 SuperMini。
2. 旋钮：三脚旋钮编码器，引脚为 `A`、`C`、`B`。
3. 显示：0.96 寸 OLED IIC 模块，照片丝印为 `GND VCC SCL SDA`。
4. 执行器：SG90 类小舵机。
5. 点击头：固定在舵机摆臂上的导电海绵、铜箔小片或触控笔头。

这个版本不再让导电海绵一直贴着屏幕，而是用舵机做真实的“抬起/按下”动作。

## 2. 面包板总思路

面包板电源轨这样用：

| 面包板电源轨 | 接什么 | 用途 |
|---|---|---|
| 红色 `+` 轨 | ESP32 `3V3` | 给 OLED 供电 |
| 蓝色/黑色 `-` 轨 | ESP32 `GND` | 所有模块共地 |

舵机红线不要接 `3V3`，要接 `5V/VBUS` 或外部 5V 电源。

## 3. 推荐 GPIO 分配

| 功能 | ESP32-C3 GPIO | 连接对象 |
|---|---:|---|
| 舵机信号 | GPIO3 | 舵机黄线/橙线 |
| 旋钮 A 相 | GPIO4 | 旋钮 `A` |
| 旋钮 B 相 | GPIO5 | 旋钮 `B` |
| OLED 数据 | GPIO8 | OLED `SDA` |
| OLED 时钟 | GPIO9 | OLED `SCL` |

旋钮 `C` 接 `GND`。

## 4. 完整接线表

### 4.1 ESP32 到面包板电源轨

| ESP32-C3 | 面包板 |
|---|---|
| `3V3` | 红色 `+` 电源轨 |
| `GND` | 蓝色/黑色 `-` 电源轨 |

### 4.2 OLED IIC 显示屏

| OLED 引脚 | 接到 |
|---|---|
| `GND` | 蓝色/黑色 `GND` 电源轨 |
| `VCC` | 红色 `+3V3` 电源轨 |
| `SCL` | ESP32 `GPIO9` |
| `SDA` | ESP32 `GPIO8` |

### 4.3 三脚旋钮编码器

| 旋钮模块 | 接到 |
|---|---|
| `A` | ESP32 `GPIO4` |
| `C` | 蓝色/黑色 `GND` 电源轨 |
| `B` | ESP32 `GPIO5` |

如果旋钮方向反了，比如顺时针变成降档，把 `A` 和 `B` 两根线对调即可。

### 4.4 舵机

以 SG90 类小舵机为例：

| 舵机线 | 接到 |
|---|---|
| 红线 | ESP32 `5V/VBUS`，或外部 5V 电源正极 |
| 棕线/黑线 | ESP32 `GND`，或外部 5V 电源负极 |
| 黄线/橙线 | ESP32 `GPIO3` |

如果用外部 5V 电源给舵机供电，必须共地：

```text
外部 5V 电源负极 -> ESP32 GND
舵机棕/黑线     -> 外部 5V 电源负极
舵机红线        -> 外部 5V 电源正极
舵机黄/橙线     -> ESP32 GPIO3
```

## 5. 面包板俯视图

```text
面包板电源轨
+3V3  ========================================================== 红色 +
GND   ========================================================== 蓝色 -

 [ESP32-C3 SuperMini]
   3V3  -------------------------------> +3V3 轨
   GND  -------------------------------> GND 轨

   GPIO8 ------------------------------> OLED SDA
   GPIO9 ------------------------------> OLED SCL

   GPIO4 ------------------------------> 旋钮 A
   GPIO5 ------------------------------> 旋钮 B

   GPIO3 ------------------------------> 舵机黄/橙信号线
   5V/VBUS ----------------------------> 舵机红线
   GND --------------------------------> 舵机棕/黑线

   VCC  -------------------------------> +3V3 轨
   GND  -------------------------------> GND 轨
   SDA  -------------------------------> GPIO8
   SCL  -------------------------------> GPIO9

 [旋钮编码器]
   A    -------------------------------> GPIO4
   C    -------------------------------> GND 轨

 [舵机]
   红线 -------------------------------> 5V/VBUS
   棕/黑 ------------------------------> GND
   黄/橙 ------------------------------> GPIO3
```

## 6. 操作逻辑

| 档位 | 状态 | 舵机按下时长 | 舵机抬起间隔 | 理论频率 |
|---:|---|---:|---:|---:|
| 0 | 停止 | 0ms | 0ms | 0 次/秒 |
| 1 | 运行 | 100ms | 250ms | 约 3 次/秒 |
| 2 | 运行 | 90ms | 160ms | 约 4 次/秒 |
| 3 | 运行 | 80ms | 120ms | 约 5 次/秒 |
| 4 | 运行 | 70ms | 90ms | 约 6 次/秒 |
| 5 | 运行 | 60ms | 70ms | 约 8 次/秒 |

上电默认 `Grade 0`，舵机保持抬起。转到 `Grade 1-5` 后，舵机按当前档位上下点击。

## 7. 舵机安装建议

1. 先不要把点击头压在手机上，空载看舵机摆动方向。
2. 把导电海绵固定在舵机摆臂末端。
3. 让 `Grade 0` 时海绵离开屏幕 1-3mm。
4. 让运行时海绵轻轻点到屏幕，不要重压。
5. 如果按下太深或太浅，修改代码里的 `SERVO_UP_US` 和 `SERVO_DOWN_US`。

## 8. 安全检查

通电前确认：

1. OLED 的 `VCC/GND` 没有接反。
2. 舵机红线没有接到 `3V3`。
3. 舵机 GND 和 ESP32 GND 共地。
4. 舵机信号线接 `GPIO3`。
5. 如果 ESP32 重启、OLED 闪烁、舵机乱抖，说明供电不够，改用外部 5V 给舵机。
